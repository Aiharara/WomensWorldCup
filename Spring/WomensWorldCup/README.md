## 如何开始?

1. 安装Maven环境. 官网下载(https://maven.apache.org/download.cgi) 记得加入path环境中.

2. 在这个readme所在的文件夹运行以下指令:

   安装依赖

   ```bash
   mvn clean install
   ```

   运行后端

   ```bash
   mvn spring-boot:run
   ```
